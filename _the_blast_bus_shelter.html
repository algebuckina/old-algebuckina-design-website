The Blast Bus Shelter

The Blast Bus Shelter was a group project to build a quick construct bus shelter for Anglicare Villages in Castle Hill. The brief requested was to use sheet metal for the some or all of the build. The group, Bus Boys, chose to create a design that harked back to 60s angular design and a colour scheme that fit the buildings of the area. The initial Solidworks file was critisized for not being constructable, so Bradley remade the enitre thing overnight, with fixing hardware and all, improving the project mark drastically. The model was created by Bradley with hand crafted and 3D printed components. This model was so good, it was featured in as the best in class at Widevision 2020.
